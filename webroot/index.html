RegexStateMachine GitHub Source Code

Converts Regular Expressions into NFA and DFA FSMs, and Test Strings
NFA NFA (Nondeterministic Finite Automata) is a type of finite-state machine where a character or unicode can have more than one state transition to go to. This NFA uses epsilon (ɛ) to allow a state to move to any other ɛ, including through multiple ɛ.
DFA DFA (Deterministic Finite Automata) is a type of finite-state machine where all ɛ are removed to have a char or unicode move to ONLY 1 state. The powerset construction is used to convert the NFA to a DFA.
DFA Minimized This DFA has redundant or equivalent states merged together, and it behaves exactly as the non-minimized DFA. Hopcroft's algorithm has been applied to minimize the states of a DFA.
